> Snapdragon utility for creating a new AST node in custom code, like plugins.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save snapdragon-node
```

## Usage

```js
var snapdragon-node = require('snapdragon-node');
```
